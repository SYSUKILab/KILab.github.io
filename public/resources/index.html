基准测试 (Benchmarks) #

KG-Benchmark Suite #

我们开发了一套综合的知识图谱基准测试套件，涵盖多个核心任务：

实体链接 (Entity Linking) #

  • 数据集: 5个领域，10万+标注样本
  • 评估指标: Precision, Recall, F1-Score
  • 基线模型: 10个SOTA模型对比

关系抽取 (Relation Extraction) #

  • 数据集: 中英双语，50个关系类型
  • 评估指标: Micro/Macro F1, AUC
  • 基线模型: 15个经典和最新模型

知识推理 (Knowledge Reasoning) #

  • 数据集: 多跳推理，复杂查询
  • 评估指标: MRR, Hits@K
  • 基线模型: 8个图神经网络模型

GitHub 在线评估

开源工具 (Tools) #

KG-Toolkit #

一站式知识图谱工具包
集成了知识抽取、推理、可视化等功能的Python工具包 
# 安装
pip install kg-toolkit

# 使用示例
from kg_toolkit import KnowledgeGraph, EntityExtractor

# 创建知识图谱
kg = KnowledgeGraph()

# 实体抽取
extractor = EntityExtractor(model='bert-base-chinese')
entities = extractor.extract("张三是中山大学的教授")

功能特性:

  • 🚀 多种预训练模型支持
  • 📊 可视化知识图谱
  • 🔧 灵活的API接口
  • 📚 详细的文档和教程

GitHub PyPI
