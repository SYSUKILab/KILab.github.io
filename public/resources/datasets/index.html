Overview #

The Knowledge Intelligence Lab maintains and shares various high-quality datasets to support research in knowledge graphs, natural language processing, and machine learning. Our datasets are openly available to the research community.

Knowledge Graph Datasets #

CN-Academic-KG #

A comprehensive Chinese academic knowledge graph containing over 1 million entities and relationships across multiple disciplines.

GitHub

MultiModal-KG #

A multimodal knowledge graph integrating textual, visual, and numerical information with 500K+ entities.

GitHub

NLP Datasets #

FewShot-RE-ZH #

Chinese few-shot relation extraction dataset with 20 relation categories and limited training samples.

GitHub

Dialogue-KG #

Knowledge graph built from real dialogues for evaluating dialogue system reasoning capabilities.

GitHub

Usage Guidelines #

All datasets are released under appropriate licenses for academic and research purposes. Please cite our papers when using these datasets in your research.
